# Me 页面优化设计文档

## 概述

优化 Me 页面，在保留原有内容框架的基础上，融入 INFP 性格特质，提升视觉设计感和艺术美感。

## 设计目标

- 保留原有内容：学习思维、合作原则、价值观、当前状态
- 融入 INFP 气质：安静、内省、留白、感受
- 提升视觉设计感：渐变色块、呼吸感、艺术排版
- 保持彩蛋式设计：不直接说 INFP，让有心人发现

## 视觉设计

### 渐变色块（呼吸感）

页面顶部添加一个缓慢流动的渐变色块，作为页面的视觉定调：

- 位置：section-header 下方，内容区上方
- 大小：占页面约 15-20% 高度
- 颜色：深色系渐变（深蓝 → 深紫 → 深青），与主站深色背景融合
- 效果：CSS 渐变动画，缓慢流动，周期约 8-10 秒
- INFP 彩蛋：渐变色块右下角藏着 `INFP` 四个字母，颜色与背景几乎一致（opacity: 0.08），hover 时略微显现

### 留白调整

- 卡片间距加大（gap 从 1px 增加到 2px）
- 卡片内边距增加（padding 从 clamp(32px, 5vw, 48px) 增加到 clamp(40px, 6vw, 56px)）
- section-header 与内容区的间距增加
- 整体呼吸感更强

### 排版调整

- 关键词字体更大，间距更宽
- 小字独白更小，更安静
- 保持现有的字体系统（Instrument Serif, DM Sans, JetBrains Mono）

## 技术实现

### CSS 动画

```css
@keyframes breathe {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}

.me-breath-gradient {
  background: linear-gradient(135deg, #0a1628, #1a0a28, #0a2818);
  background-size: 400% 400%;
  animation: breathe 10s ease-in-out infinite;
}
```

### 响应式

- 移动端：渐变色块高度减小，留白适当减少
- 保持单屏叙事，不产生滚动（或极少滚动）

## 与现有风格的统一

- 保持深色背景（#0a0a0a）
- 保持金色强调（#d4af37）
- 保持字体系统（Instrument Serif, DM Sans, JetBrains Mono）
- 保持 Neo-Brutalist 的边框风格
- 渐变色块是新增的柔和元素，与现有的硬边框形成对比，增加层次感

## 成功标准

- 访客能感受到页面的安静氛围
- 保留的内容依然清晰可读
- INFP 彩蛋不突兀，有心人会发现
- 整体设计感提升，但不喧宾夺主
