# 个人网站设计文档

## 概述

这是一个个人网站项目，目标是创建一个高端、极简、优雅的个人展示网站，类似高端展厅的感觉。网站将展示个人作品、兴趣爱好、技能、音乐、自证预言等内容。

## 设计风格

### 整体风格

- **极简主义**：大量留白，干净排版，专注于内容本身
- **高端感**：像在一个高端展厅，美丽、克制、优雅
- **参考**：李笑来的个人网站风格，但做出自己的特色

### 颜色方案

- **主色调**：黑色背景（#0a0a0a）
- **强调色**：金色点缀（#d4af37）
- **文字色**：白色（#fff）和半透明白色（rgba(255,255,255,0.6)）

### 字体

- **风格**：优雅衬线字体
- **中文字体**：宋体、楷体等衬线字体
- **英文字体**：Times New Roman、Georgia等衬线字体
- **名字显示**：用"觉"或"Yijian Liu"代替真名

## 网站结构

### 导航栏

- **位置**：页面顶部
- **样式**：纯文字导航，无背景
- **布局**：左侧名字"觉"，右侧板块名称
- **板块顺序**：Work → Offer → Passion → Hobby → Prediction

### 页面结构

- **类型**：导航栏 + 独立页面
- **首页**：简单淡入动画后显示导航
- **各板块**：独立页面，点击导航切换

## 板块设计

### Work 板块

- **内容**：项目列表
- **布局**：时间线/故事式
- **展示**：按时间顺序展示项目，每个项目有标题、描述、链接

### Offer 板块

- **内容**：能为别人提供的帮助和服务
- **布局**：列表式
- **展示**：列出可提供的服务和价值，每个服务有标题和描述

### Passion 板块

- **内容**：兴趣爱好 + 生活记录
- **布局**：精选时刻
- **展示**：只展示最值得记录的时刻，追求质量而非数量

### Hobby 板块

- **内容**：Books + Skills + Music 合并
- **布局**：三个子板块，标签页切换
- **展示**：
  - Books：书评和动态交互
  - Skills：技能熟练度展示
  - Music：音乐交互

### Prediction 板块

- **内容**：自证预言、未来的我
- **布局**：时间线式
- **展示**：按时间展示预言，如"2025年我要..."，"2026年我要..."

## 页面底部

### Golden Sentence

- **位置**：每个页面底部
- **内容**：人生金句、道理、指南
- **样式**：优雅排版，与整体风格统一

### 联系方式

- **位置**：页面底部（Footer）
- **内容**：邮箱、GitHub、微信等
- **样式**：简洁排列，不喧宾夺主

## 首页动画

### 动画效果

- **类型**：简单淡入 + 导航
- **过程**："觉"字淡入，然后显示导航栏
- **时长**：适中，不要太快也不要太慢
- **风格**：优雅、克制，符合整体高端感

## 响应式设计

### 设计优先级

- **桌面端（1200px+）**：完整体验，所有动效和交互
- **平板（768px-1199px）**：布局微调，保持主要功能
- **手机（<768px）**：导航变汉堡菜单，内容单列，简化动效

### 适配策略

- 以桌面端为主要设计对象
- 移动端做响应式适配
- 确保手机用户能正常浏览

## 技术选型

### 技术栈

- **HTML**：页面结构
- **CSS**：样式设计
- **JavaScript**：交互效果

### 开发方式

- **纯手写**：从零开始写每一行代码
- **理由**：
  1. 完全掌控
  2. 学习最深入
  3. 无依赖
  4. 长期可维护

## 开发计划

### 第一阶段：基础骨架（今天）

1. 创建项目文件夹和基础文件
2. 实现首页淡入动画
3. 搭建导航栏框架
4. 创建各板块空白页面
5. 实现页面跳转

### 第二阶段：样式美化（近期）

1. 实现黑底+金色点缀的配色
2. 应用优雅衬线字体
3. 设计页面底部的Golden Sentence和联系方式
4. 优化各板块的布局和间距
5. 实现响应式适配

### 第三阶段：内容填充（中期）

1. Work板块：添加项目列表
2. Passion板块：添加精选时刻
3. Hobby板块：实现标签页切换，添加内容
4. Prediction板块：添加时间线式预言
5. 添加Golden Sentence内容

### 第四阶段：交互增强（远期）

1. 添加滚动动画效果
2. 实现Hobby板块的动态交互
3. 优化移动端体验
4. 考虑NFC卡片功能

## 特殊功能

### NFC卡片

- **目标**：制作成NFC卡片，手机一扫就能访问
- **实现**：后期考虑，先完成网站本身

### 动态交互

- **Books**：动态交互和书评
- **Skills**：技能熟练度展示
- **Music**：音乐交互

## 成功标准

### 短期目标

- 能打开一个能用的网站
- 能点击导航跳转
- 视觉上高端、优雅

### 长期目标

- 成为一个可以长期迭代的个人展示平台
- 完全掌控，能自己修改任何部分
- 成为未来几十年都可以使用的个人品牌

## 注意事项

### 设计原则

- 保持极简，不要过度设计
- 高端感来自克制，不是堆砌
- 内容为王，形式服务于内容

### 开发原则

- 从基础开始，逐步完善
- 每一步都要理解原理
- 不要急于求成，慢慢来

### 维护原则

- 代码清晰易懂
- 方便后续修改
- 长期可维护
