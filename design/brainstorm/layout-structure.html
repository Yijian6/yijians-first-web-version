页面结构怎么安排？

极简风格下，内容的组织方式很重要

A

单页长滚动

所有板块在一个页面内，向下滚动浏览。开场动画 → 个人简介 → Books → Skills → Passion → Music → 金句 → Work → 联系方式

优点

  • 浏览体验连贯
  • 实现简单
  • 移动端友好

缺点

  • 页面会很长
  • 不方便跳转到特定板块
B

导航栏 + 独立页面

顶部导航栏，每个板块是独立页面。点击导航切换，类似传统网站结构。

优点

  • 结构清晰
  • 便于扩展
  • 可独立更新各板块

缺点

  • 需要多页面管理
  • 页面切换有加载
C

首页入口 + 板块页面

首页是精美的入口（你的名字+动画），点击进入后是各板块的导航页，再点击进入具体内容。

优点

  • 仪式感强
  • 首页冲击力大
  • 层次分明

缺点

  • 多一次点击才能看到内容
  • 实现复杂度中等
