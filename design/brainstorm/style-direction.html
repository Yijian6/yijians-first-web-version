你的个人网站想要什么风格？

选择一个整体设计方向，我们会在此基础上细化

A

极简主义

大量留白，干净的排版，专注于内容本身。类似苹果官网、李笑来的风格。

优点

  • 永不过时
  • 加载快
  • 易于维护

缺点

  • 可能显得单调
  • 需要好的内容支撑
B

科技感 / 未来感

深色背景，霓虹色点缀，动态粒子效果，代码风格字体。适合展示技术能力。

优点

  • 视觉冲击力强
  • 符合计算机专业形象

缺点

  • 可能喧宾夺主
  • 实现复杂度高
C

温暖个人化

手写风格字体，温暖色调，照片墙，日记式排版。突出个人特色和故事感。

优点

  • 亲切感强
  • 容易产生共鸣

缺点

  • 可能不够专业
  • 照片质量要求高
D

杂志 / 作品集风格

网格布局，大图展示，类似Behance或Dribbble。适合展示作品和项目。

优点

  • 视觉丰富
  • 适合展示作品

缺点

  • 需要大量素材
  • 移动端适配复杂
