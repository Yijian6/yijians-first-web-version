开发顺序

一步一步来，先做出能用的，再慢慢完善

第一阶段：基础骨架（今天）

目标：做出一个能打开、能点击的网站

具体任务：

1. 创建基础HTML文件结构

2. 实现首页淡入动画

3. 搭建导航栏框架

4. 创建各板块的空白页面

5. 实现页面间跳转

产出：一个能打开、能点击导航、能在页面间跳转的网站

第二阶段：样式美化（近期）

目标：让网站看起来高端、优雅

具体任务：

1. 实现黑底+金色点缀的配色

2. 应用优雅衬线字体

3. 设计页面底部的Golden Sentence和联系方式

4. 优化各板块的布局和间距

5. 实现响应式适配（桌面优先）

产出：视觉上高端、优雅的网站

第三阶段：内容填充（中期）

目标：填入真实内容

具体任务：

1. Work板块：添加项目列表

2. Passion板块：添加精选时刻

3. Hobby板块：实现标签页切换，添加Books/Skills/Music内容

4. Prediction板块：添加时间线式的自证预言

5. 添加Golden Sentence内容

产出：有真实内容的完整网站

第四阶段：交互增强（远期）

目标：添加动态效果和交互

具体任务：

1. 添加滚动动画效果

2. 实现Hobby板块的动态交互

3. 优化移动端体验

4. 考虑NFC卡片功能

产出：交互丰富的完整网站
