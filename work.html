我的作品

这里记录着一些我做过的东西。它们并不完美，也还在继续生长。

10圈年轮 since 2026

  • 拾序记账

    《拾序记账》不想让记账变成一套复杂的财务管理，而是想把模糊的消费感受，变成简单易懂的生活秩序。从金额到去向，再到落笔，三步记完；统计只回答钱去了哪里，不做评价，也不制造焦虑。所有的数据都留在自己的设备里，清楚，也安心——如果你也想知道钱去了哪里，欢迎试试看。

    立即体验

  • 今日三格

    《今日三格》每天只问你最重要的三件事。填满三格，今天就有了一个清楚的答案；少做一件，也不必被进度和计划追着走。它没有复杂的任务层级，只是用最简单的方式，让你确定今天真正重要的事，从而保持专注。登录后可以跨设备同步，打开就能开始。

    立即体验

  • 造像 ATELIER

    《造像 ATELIER》想做的，是让每个人都能更轻松地尝试不同的自己。上传几张照片，选一个喜欢的气质方向，AI 会基于经过反复打磨的创意方案和提示词，为你生成头像、封面或简历照。目前，它已经能够在进行创意表达的同时，尽可能保留人物的面部特征。我还在继续调整它，希望生成的照片既保留你的样子，也能呈现你想尝试的气质。

    立即体验

  • AI 学习策展室

    《AI 学习策展室》想解决一个很简单的问题：想了解 AI，却不知道从哪里开始、今天该学什么？它会根据持续积累的学习档案，每天策划一节面向 AI 初学者的课。每天中午，新的内容会自动生成并上线。希望它能为正在学习 AI 的人，提供一些方向和启发。

    立即体验

  • AI 工具雷达

    面对越来越多的 AI 工具，真正让人困惑的往往不是没有选择，而是不知道哪些值得开始。这里每天整理一份简明的 AI 工具参考，只收录真正免费、开源或容易上手的工具，尽量把新闻噪音挡在外面。希望它能帮刚开始接触 AI 的人，找到下一件值得试试的工具。

    立即体验

  • Type Ledger

    《Type Ledger》来自一个很小的好奇：我们每天在键盘上，到底留下了多少文字？它是一款 Windows 桌面工具，会安静地记录每天的打字量，让那些不容易察觉的输入慢慢变得可见。所有数据只保存在本地，不上传，也不需要联网。如果你也好奇自己一天写了多少字，可以用它看看。

    GitHub

  • 审美日课

    《审美日课》从一个很朴素的问题开始：看到喜欢的画面时，除了“好看”，还能怎样说得更具体？它每天拆解一个审美概念，从构图、色彩到视觉层级，慢慢把模糊的感受变成可以理解和使用的语言。希望这些日积月累的观察，也能帮助正在使用 AI 创作的人，更清楚地表达自己想要什么。

    立即体验

  • 音频批量转文字

    《音频批量转文字》是为了解决一个很实际的小麻烦：手里有一堆直播、访谈或播客音频，想整理成文字，却不想一个个上传、一个个等。把文件一次放进去，它就会自动帮你转成文字，省下重复操作，也让后面的阅读和整理更容易。希望它能让那些差点被放下的音频，重新变成可以阅读、搜索和使用的内容。

    GitHub

  • Minimal WPM Tracker

    《Minimal WPM Tracker》来自一个简单的问题：我读书到底有多快？它记录每次阅读的速度，用一条清楚的趋势帮你看见自己的阅读节奏。不做复杂的分析，也不急着给出结论，只留下几次阅读之间慢慢发生的变化。因为想知道自己读得多快，所以做了一个刚好够用的工具。

    GitHub

  • 个人网站 — 觉

    这是我为自己做的一块线上空间。这里放着我的想法、作品，也记录着我如何一步步把它们做出来。其中不少页面，是我和 AI 一起反复讨论、修改出来的。从第一版到现在，它经历过几次推翻和重来，仍然没有完成。接下来我会继续用它记录新的尝试。
